# Agentic Orchestration Stack

Repository for building and refining a practical agentic workflow.

## Builder-Organizer init

To initialize the Builder-Organizer in a new Codex session:

```text
Initialize from `agentic/prompts/init-builder-organizer.md`. Do not start implementation.

Important:

task opening and task execution are separate steps
new tasks start as draft
implementation begins only after explicit approval moves a task to ready
```
