# workflow-v2

This folder contains the experimental `workflow-v2` agent collaboration docs and examples.

## Files

- `AGENT_WORKFLOW.md`
  - main workflow spec
- `state.example.yaml`
  - example global coordination state
- `tasks.example.yaml`
  - example task registry
- `handoffs/TS-EXAMPLE.md`
  - example task handoff entry

## Location

```text
agentic/experiments/workflow-v2/AGENT_WORKFLOW.md
```

Example handoff file:

```text
agentic/experiments/workflow-v2/handoffs/TS-EXAMPLE.md
```

## Notes

- This folder is an experiment, not the promoted default workflow location.
- If this workflow becomes the default later, it can move to a cleaner path such as `agentic/workflow/`.
